## Descripción
Este proyecto consiste en un generador de imágenes por capas desarrollado en C++. El sistema permite cargar capas, imágenes y usuarios desde archivos de texto, generar imágenes mediante la superposición de capas y visualizar estructuras de memoria mediante Graphviz.

## Archivos principales
- main.cpp
- proyecto.exe
- capas.cap
- imagenes.im
- usuarios.usr
- Manual_Usuario_Proyecto_1_Generador_Capas.docx
- Manual_Tecnico_Proyecto_1_Generador_Capas.docx

## Requisitos
- Compilador g++ / MinGW
- Graphviz instalado
- Sistema operativo Windows
